Grupo de Botões

Agrupe botões juntos em uma única linha

Dicas de ferramentas precisam de uma configuração personalizada

Ao utilizar dicas de ferramentas em elementos como .btn-group, você tera que especificar a opção container: 'body' para evitar efeitos indesejados.

Lembre-sede colocar um role e forneceruma etiqueta

Para que os botões agrupados sejam transmitidos da maneira correta, um atributo role precisa ser fornecido. para um grupo de botões você ira utilizar role="group", enquanto em toolbars (barra de ferramentas) utilizará role="toolbar"

uma exceção são grupos que contém apenas um único controle, como quando utilizamos o elemento buttonou um dropdown

Grupos de barras de ferramentas devem ter uma etiqueta explícita, pois a maioria das tecnologias assistivas querem caso contrário não os anunciam, apesar da preseça do atributo role correto. Em um exemplo fornecido aqui, nós usamos aria-label, mas alternativas tal como aria-labelledbypodem também ser usadas.
